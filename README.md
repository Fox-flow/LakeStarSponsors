# Lakestar Sponsors Website

## Structure

- `index.html` - Main HTML file
- `css/style.css` - Stylesheet
- `js/main.js` - JavaScript for interactivity
- `images/` - Image assets (logos, team photos, icons)
  - `team/` - Team member photos
  - `logos/` - Portfolio company logos
  - `icons/` - Sector icons

## Features

- Responsive navigation with dropdown menu
- Hero section
- Who We Are section
- Sector Focus grid
- Investment Criteria
- Team member grid
- Portfolio with filtering (Active/Exited/Prior Experience)
- News section
- Contact information
- Smooth scrolling navigation
